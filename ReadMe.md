---

**Logged-In User Can:**

- **See All His Tasks.**
- **Change Task Status.**

## Administration.

---

**Admin Can:**

- **Create New Task.**
- **Delete/Update Task.**
- **See All Tasks/Task`s with specific status.**
- **Add new Task to User.**
- **See all Users and Tasks of Specific user.**
- **See most Productive Users.**
